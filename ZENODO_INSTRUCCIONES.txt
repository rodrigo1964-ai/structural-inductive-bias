===================================

1. GENERAR EL ZIP
   cd /home/rodo/15Paper
   python build_zenodo.py

   Esto crea: structural-inductive-bias-v1.0.zip

2. IR A ZENODO
   https://zenodo.org
   → Log in con GitHub (o crear cuenta)

3. SUBIR EL ZIP
   Arrastrá structural-inductive-bias-v1.0.zip al área de upload

NOTAS
=====
- El DOI es permanente e inmutable. Una vez publicado, no se puede borrar.
- Si necesitás corregir algo, Zenodo permite crear una "New version"
  (nuevo DOI, pero linkado al anterior).
- El ZIP queda almacenado en los servidores del CERN. No depende de
  GitHub ni de tu máquina.
- Tamaño máximo por archivo: 50 GB (sobra).
- El upload es gratuito y sin límite de storage.
