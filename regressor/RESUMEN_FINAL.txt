================================================================================
ORDEN DE CONVERGENCIA
================================================================================

Análisis experimental determinó:

  Orden de convergencia: p ≈ 0.91

El error disminuye aproximadamente como O(T^0.91) ≈ O(T)

REGLA PRÁCTICA:
  • Reducir T a la mitad    → Error mejora ~1.9×
  • Reducir T en 10 veces   → Error mejora ~8-10×

================================================================================
RECOMENDACIONES PRÁCTICAS
================================================================================

SELECCIÓN DEL PASO TEMPORAL:
─────────────────────────────

  Sistema Lineal                  → n ≈ 500-1,000    (T ≈ 0.01-0.02)
  Sistema No Lineal Moderado      → n ≈ 1,000-2,000  (T ≈ 0.005-0.01)
  Sistema Muy No Lineal           → n ≈ 10,000-20,000 (T ≈ 0.001-0.003)
  Sistema Caótico / Muy Sensible  → n ≥ 20,000       (T < 0.001)


USAR REGRESOR HOMOTÓPICO:
──────────────────────────

  ✓ Ecuación conocida con derivadas calculables
  ✓ Alta eficiencia computacional requerida
  ✓ Implementación en sistemas embebidos/microcontroladores
  ✓ Control en tiempo real
  ✓ Simplicidad de código importante
  ✓ Se acepta ajustar paso temporal según sistema


USAR RK4 O MÉTODOS SUPERIORES:
───────────────────────────────

  ✓ Máxima precisión con mínimo ajuste
  ✓ Derivadas difíciles de calcular
  ✓ Sistema desconocido a priori
  ✓ Se prefiere método más "automático"
  ✓ Costo computacional no es crítico

================================================================================
FORTALEZAS DEL REGRESOR
================================================================================

1. ✅ PRECISIÓN competitiva con RK4 en todos los tipos de sistemas
2. ✅ MUY EFICIENTE: 2-235 ms para 500-20,000 puntos
3. ✅ SIMPLE: Solo 3 correcciones por paso
4. ✅ ESCALABLE: Ajustar T mejora el error predeciblemente
5. ✅ DETERMINISTA: Costo computacional fijo
6. ✅ MEMORIA MÍNIMA: Ideal para embebidos
7. ✅ CÓDIGO SIMPLE: Fácil de implementar y mantener

================================================================================
CONSIDERACIONES
================================================================================

1. ⚠️ REQUIERE DERIVADAS: f, f', f'', f'''
   → Solución: Usar derivadas numéricas si no hay analíticas

2. ⚠️ SENSIBLE AL PASO en sistemas muy no lineales
   → Solución: Usar T más pequeño (más puntos)
   → Regla: Reducir T a la mitad mejora error ~1.9×

3. ⚠️ CONVERGENCIA más lenta que RK4: O(T^0.91) vs O(T⁴)
   → Impacto: Para misma precisión, regresor necesita T más pequeño
   → Compensación: Costo por paso es menor
